## Key Technical Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Framework | Astro | Static site, Claude Code friendly |
| Styling | Tailwind CSS | Utility-first, fast development |
| Animations | CSS + vanilla JS | No heavy libraries, performant |
| Icons | Lucide | Lightweight, works with Astro |
| Deployment | GitHub Pages | Free, simple |
