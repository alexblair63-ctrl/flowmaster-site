## 4.2 Add Animation to Global CSS

Add to **src/styles/global.css** (in @layer utilities):
```css
@layer utilities {
  .text-gradient {
    @apply bg-clip-text text-transparent bg-gradient-to-r from-brand-red to-brand-red-bright;
  }
  
  .animate-fade-in-up {
    animation: fade-in-up 0.8s ease-out forwards;
  }
}

@keyframes fade-in-up {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

## Acceptance Criteria

- [ ] Hero takes full viewport height (min-h-screen)
- [ ] Content is vertically and horizontally centered
- [ ] Logo/brand name displays prominently
- [ ] "Edit as you record" headline is large and bold
- [ ] Description text is visible but muted (gray)
- [ ] Two CTA buttons present: "Download Beta" (primary) and "See How It Works" (secondary)
- [ ] "Free during beta • Windows 10/11" text below buttons
- [ ] All elements animate in with staggered delays
- [ ] Scroll indicator arrow at bottom
- [ ] Hero has id="hero" for nav scroll detection
- [ ] Smooth, professional feel
