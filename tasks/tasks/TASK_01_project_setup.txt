# Task 01: Project Setup & Structure

## Goal
Set up the Astro project with Tailwind CSS and create the folder structure.

---

## 1.1 Initialize Project (if not done)

If starting fresh:
```bash
cd ~/Desktop
npm create astro@latest flowmaster-site
# Choose: minimal (empty) template
# Yes to dependencies
# Yes to git
cd flowmaster-site
```

---

## 1.2 Add Tailwind CSS

```bash
npx astro add tailwind
```

When prompted, accept the default options.

---

## 1.3 Create Folder Structure

Create the following directories and files:

```
flowmaster-site/
├── src/
│   ├── layouts/
│   │   └── BaseLayout.astro
│   ├── components/
│   │   ├── Header.astro
│   │   ├── Footer.astro
│   │   ├── Button.astro
│   │   └── animations/
│   │       └── .gitkeep
│   ├── pages/
│   │   ├── index.astro
│   │   ├── features.astro
│   │   ├── future.astro
│   │   ├── about.astro
│   │   ├── download.astro
│   │   └── docs/
│   │       ├── index.astro
│   │       └── obs-setup.astro
│   ├── styles/
│   │   └── global.css
│   └── assets/
│       └── .gitkeep
├── public/
│   └── favicon.svg
├── astro.config.mjs
├── tailwind.config.mjs
└── package.json
```

Commands to create structure:
```bash
mkdir -p src/layouts src/components/animations src/pages/docs src/styles src/assets
touch src/layouts/BaseLayout.astro
touch src/components/Header.astro
touch src/components/Footer.astro
touch src/components/Button.astro
touch src/components/animations/.gitkeep
touch src/pages/features.astro
touch src/pages/future.astro
touch src/pages/about.astro
touch src/pages/download.astro
touch src/pages/docs/index.astro
touch src/pages/docs/obs-setup.astro
touch src/styles/global.css
touch src/assets/.gitkeep
touch public/favicon.svg
```

## 1.4 Basic Placeholder Files

**src/layouts/BaseLayout.astro:**
```astro
---
interface Props {
  title: string;
  description?: string;
}

const { title, description = "Edit as you record - Video editing automation for content creators" } = Astro.props;
---

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta name="description" content={description} />
    <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
    <title>{title} | Flowmaster</title>
  </head>
  <body>
    <slot />
  </body>
</html>
```

**src/pages/index.astro:**
```astro
---
import BaseLayout from '../layouts/BaseLayout.astro';
---

<BaseLayout title="Home">
  <main>
    <h1>Flowmaster - Coming Soon</h1>
    <p>Edit as you record</p>
  </main>
</BaseLayout>
```

**public/favicon.svg:**

## 1.5 Verify Setup

```bash
npm run dev
```

Open http://localhost:4321 and confirm:
- Page loads without errors
- "Flowmaster - Coming Soon" appears
- Favicon shows in browser tab
