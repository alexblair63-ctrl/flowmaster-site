 Task 7: LemonSqueezy Account & Product Setup

 Objective

 Set up LemonSqueezy account, create the
 Flowmaster Suite product, and configure
 license key generation.

 Type: Manual setup task (no code changes)
 Estimated Time: 30-60 minutes
 Dependencies: None

 ---
 Setup Checklist

 Step 1: Create LemonSqueezy Account

 - Go to https://app.lemonsqueezy.com/register
 - Sign up with business email
 - Verify email address
 - Complete account setup

 Step 2: Configure Store

 - Go to Settings → Store
 - Set store name (e.g., "Flowmaster" or
 "Alenzie Solutions")
 - Upload Flowmaster logo (optional)
 - Set support email
 - Note your Store ID (visible in URL or
 settings)

 Step 3: Create Product

 - Go to Products → Create Product
 - Name: Flowmaster Suite (or Flowmaster Suite
 Beta for beta)
 - Description: "Video editing automation
 toolkit for content creators"
 - Price: $0.00 for beta (can change later)
 - Click Create

 Step 4: Enable License Keys

 - In product settings, find License Keys
 section
 - Enable license keys
 - Set activation limit: 2 (devices per
 license)
 - Note your Product ID

 Step 5: Generate API Key

 - Go to Settings → API
 - Click Create API Key
 - Name it (e.g., "Flowmaster App")
 - Copy and save the API key securely

 Step 6: Test (Optional)

 - Create a test order/license manually
 - Verify license key format looks correct
